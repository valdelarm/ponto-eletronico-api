API do sistema de ponto eletrônico inteligente com Java 8 e Spring Boot para propósitos de estudos.

# Tecnologias e Frameworks utilizados:
Java 8
Spring Boot
Spring Security
TravisCI
FlyWay
JUnit test
Maven
JPA
MySQL
H2
JWT
Swagger
EhCache
